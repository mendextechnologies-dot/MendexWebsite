# **App Name**: Mendex Platform

## Core Features:

- AI-Driven Service Matching: Intelligently match users with appropriate services using AI algorithms in the OPS Marketplace. This AI will act as a tool to help suggest possible service matchings and should provide its reasoning to the end user.
- Product Showcase: Display product cards for OPS Marketplace and MendexOne CRM on the homepage with descriptions and links to dedicated pages.
- Workflow Automation: Implement workflow automation features in MendexOne CRM for streamlined business management.
- Contact Form: Allow potential customers to directly send an email to Mendex through a contact form.

## Style Guidelines:

- Primary color: Cool and calming desaturated blue (#79A4BC) to convey trust and stability.
- Background color: Light gray (#F0F4F7) for a clean and modern enterprise look.
- Accent color: Teal (#468A8F) to add a touch of sophistication.
- Body and headline font: 'Inter', a sans-serif for a clean, modern look.
- Use flat, minimalistic icons for services, features, and benefits.
- Card-based layout with clear sections for products, services, and company information.
- Subtle hover effects and transitions to enhance user engagement.
